# 标准流
* 网页的三大流派
* 中华武术的三大流派：少林，武当
* 我们的网页也有三大流派：标准流（文档流），浮动流，定位流
* 特点： 我们的标准流的特点：块级元素独占一行，行内元素一行并存（我们前面案例的特点）

        div{

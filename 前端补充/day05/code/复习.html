第一， 边框样式--语法格式 border:1px solid #css;---严格按照这个顺序来书写--规范 虚线 border:1px dashed #ccc; 扩展点线 border:1px dotted #ccc; 扩展分解写法---顺时针--规范 上边框 border-top: 1px solid #css; 左边框 border-left：1px solid #css; 下边框 border-bottom:1px solid #css; 右边框 border-right: 1px solid #css; 第二， css的层叠性：就是后写的样式会覆盖先写的样式（必须保证权量相同的情况下） 第三， css的续承性：我们的css可以实现子承父业（一般文字三属性：字号，颜色和字体) 第四， 行高 line-height 可以控制行与行之间的距离，还可以控制单行文字的垂直居中（一般用在网站导航，如果想控制网站导航文字垂直居中---高度是多少行高就给多少） 第五， 盒子模型 我们的盒子模型：具有宽度，高度，边框，外边距和内边距 外边距--margin 控制
